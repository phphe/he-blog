Vue 在组件内部更新 body 样式

24-06 2021

首先, 在模板里创建一个新元素:

<div v-html="'<style>'+styleText+'</style>'"></div>

然后设置styleText的值:

styleText: `
body {
  background: #ffe4c4;
}
`.trim()

你可以动态修改 styleText 的值, 修改后样式会自动更新. 组件卸载后样式也会被移除.
